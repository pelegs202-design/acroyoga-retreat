  <name>Task 1: Dark-first design tokens and brand accent color</name>
  <files>src/app/globals.css</files>
  <action>
Update `src/app/globals.css` to implement the dark-first brutalist theme and brand accent tokens.

The brand direction is: dark background (#0a0a0a), light foreground (#ededed), bold/physical/direct.

**Changes to make:**

1. **Remove the `@media (prefers-color-scheme: dark)` block entirely.** The app is always dark — no light mode. Set `:root` variables to dark values directly:
   ```
   --background: #0a0a0a;
   --foreground: #ededed;
   ```

2. **Add brand accent color variables in `:root`:**
   ```
   --brand: oklch(0.75 0.18 145);
   --brand-foreground: oklch(0.12 0.02 145);
   ```
   This is a vivid green accent (hue 145) — energetic, physical, high contrast against #0a0a0a background. oklch ensures perceptual consistency.

3. **Add a plain `@theme` block (NOT inside `@theme inline`)** for brand color tokens. This must be a SEPARATE block from the existing `@theme inline`:
   ```css
   @theme {
     --color-brand: var(--brand);
     --color-brand-foreground: var(--brand-foreground);
   }
   ```
   This generates `bg-brand`, `text-brand`, `border-brand` etc. utility classes.
   Do NOT add color tokens to the existing `@theme inline` block — `inline` modifier would break CSS variable runtime overrides. See research Pitfall 5.

4. **Keep the existing `@theme inline` block unchanged** — it handles font aliasing correctly.

5. **Keep the `body` rule** but ensure it still references `var(--background)` and `var(--foreground)`.

**Final file structure should be:**
```css
@import "tailwindcss";

:root {
  --background: #0a0a0a;
  --foreground: #ededed;
  --brand: oklch(0.75 0.18 145);
  --brand-foreground: oklch(0.12 0.02 145);
}

@theme {
  --color-brand: var(--brand);
  --color-brand-foreground: var(--brand-foreground);
}

@theme inline {
  --color-background: var(--background);
  --color-foreground: var(--foreground);
  --font-sans: var(--font-heebo), ui-sans-serif, system-ui, sans-serif, "Apple Color Emoji", "Segoe UI Emoji", "Segoe UI Symbol", "Noto Color Emoji";
  --font-mono: ui-monospace, "Cascadia Code", "Source Code Pro", Menlo, Consolas, "DejaVu Sans Mono", monospace;
}

body {
  background: var(--background);
  color: var(--foreground);
}
```
