All components are `"use client"` — Framer Motion and interactive state require client rendering.

**QuizEngine.tsx** — Core quiz shell component. Props:
- `questions: Question[]` — declarative question schema (type defined inline or imported)
- `quizType: 'challenge' | 'workshop'` — for analytics and DB
- `onComplete: (state: QuizState) => void` — called when final step is answered
- `storageKey?: string` — localStorage key for save/resume (optional)

Type definitions (export these):
```typescript
export type QuestionOption = {
  id: string;
  label: { en: string; he: string };
  icon?: string;
  image?: string;
  nextQuestionId?: string; // branching override
};

export type Question = {
  id: string;
  text: { en: string; he: string };
  subtitle?: { en: string; he: string };
  type: 'single-choice' | 'city-select' | 'contact';
  options?: QuestionOption[];
  defaultNextId?: string;
};

export type QuizState = {
  currentQuestionId: string;
  answers: Record<string, string>;
  direction: 1 | -1;
  sessionId: string;
  history: string[]; // stack of visited question IDs for back navigation
  contactInfo: { name: string; email: string; phone: string } | null;
};
```

Reducer actions: ANSWER (sets answer, pushes current to history, advances to next), BACK (pops history, sets direction -1), SET_CONTACT, RESTORE (rehydrate from localStorage), RESET.

On mount (useEffect): check localStorage for saved state via `storageKey`, dispatch RESTORE if found. Generate sessionId via `crypto.randomUUID()` if no saved state.

On every state change (useEffect): save to localStorage if `storageKey` provided.

Rendering: Use `AnimatePresence custom={direction} mode="wait"` wrapping a `motion.div` keyed by `currentQuestionId`. Slide variants:
```
enter: (dir) => ({ x: dir > 0 ? '100%' : '-100%', opacity: 0 })
center: { x: 0, opacity: 1, transition: { type: 'spring', stiffness: 300, damping: 30 } }
exit: (dir) => ({ x: dir < 0 ? '100%' : '-100%', opacity: 0 })
```

**CRITICAL RTL handling:** Read `dir` attribute from `document.documentElement` (or use a prop). For RTL, multiply direction by -1 so "forward" slides from left (natural reading direction). Apply: `const rtlDir = isRtl ? -direction : direction`.

Render QuizProgressBar at top. For current question: if type is 'contact', render QuizContactStep. If type is 'single-choice' or 'city-select', render QuizCard for each option. Show a Back button (if history is non-empty) that dispatches BACK.

Call `trackQuizStep` from quiz-analytics on each ANSWER dispatch. Call `trackQuizStart` on mount if no restored state.

**QuizCard.tsx** — A single visual option card. Props: `option: QuestionOption`, `onSelect: () => void`, `locale: string`. **QuizProgressBar.tsx** — Props: `current: number`, `total: number`. Render: a thin horizontal bar (h-1.5 or h-2) with bg-neutral-800 track and bg-brand fill. Width = `(current / total) * 100%`. Animate width changes with Framer Motion `motion.div` using `animate={{ width }}` with spring transition. Show "Step X of Y" text above the bar in small muted text.
