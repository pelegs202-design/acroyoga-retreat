- Import `workshopQuestions` from `@/lib/quiz/workshop-questions`
- Render `QuizEngine` with `workshopQuestions`, `quizType="workshop"`, `storageKey="quiz_workshop_state"`
- `onComplete` callback:
  1. POST to `/api/quiz/workshop` with sessionId, contactInfo, answers (group-type and group-details)

  3. Redirect to `/quiz/workshop/confirmation`
- Fire quiz events on each step (same as challenge flow)

**api/quiz/workshop/route.ts** — POST endpoint:
- Accept: `{ sessionId, name, email, phone, answers: Record<string, string> }`
- Validate with zod — answers must include group-type, group-details; optional: workshop-details (preferredDates, specialRequests)
- Insert into `quizLeads` table with quizType='workshop', resultType=null
- Send email notification to owner using Resend:
  ```typescript

           <p><strong>Name:</strong> ${name}</p>
           <p><strong>Email:</strong> ${email}</p>
           <p><strong>Phone:</strong> ${phone}</p>
           <p><strong>Group Type:</strong> ${answers['group-type'] || 'N/A'}</p>

  ```
