3-5 feature sections stacked vertically, each wrapped in ScrollReveal with staggered delays (0, 0.15, 0.3...). Each feature:
- Bold number or icon (01, 02, 03...) in text-8xl text-brand/20 as background decoration

- A card-like container with `card-hover` class

Uses HorizontalScrollSection from effects library. Content: 4-5 brutalist showcase cards (~80vw each) with:
- Platform value propositions: "Find Your Partner", "Join Jams", "30-Day Challenge", "Community Trust", "Your City"

- On mobile (below md breakpoint): render as a vertical stack of smaller cards WITHOUT drag (DraggableCard automatically falls back to static on mobile)
- All text from i18n `home.showcase` namespace

**D) `src/components/home/CTASection.tsx`** — "use client"
Bottom conversion section:
- Bold heading: "Ready to fly?" / similar from i18n
- Two magnetic CTA buttons (MagneticWrapper): primary quiz link + secondary members link
- Both buttons have `btn-press` class for tactile press feedback
- Dark background with subtle pink accent border or gradient at top

**E) `src/components/home/ScrollProgressBar.tsx`** — "use client"
Fixed position thin bar (2px) at very top of viewport (top-0, z-[60] to sit above header). Uses Framer Motion `useScroll` for page-level scrollYProgress. `scaleX` transform from 0 to 1. Background: #F472B6. Transform origin: left (or start for RTL — use `origin-[0%]` which works both directions).
